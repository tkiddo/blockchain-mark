## Day 1

密码学基础

- 哈希算法

哈希碰撞：对于两个不同的输入，进过哈希计算后得到同样的输出，则改哈希算法存在哈希碰撞。

- 加密方式：

1. 对称加密：加密和解密用同样的密钥

2. 非对称加密（公钥加密 Public-key cryptography）：加密和解密用不同密钥。

非对称加密举例： 小明在本地生成一对公私钥，把公钥发给小红，小红用公钥把信息加密后发给小明，小明收到信息后用自己的私钥解密。

签名：小明用自己的私钥给信息签名，发送给小红后，小红用私钥验证信息来自小明。
